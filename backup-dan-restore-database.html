Backup dan Restore Mysql pada linux

Full backup pada mysql/mariadb akan membackup databases dan user beserta permissionnya kedalam bentul file sql. Cara paling mudah dilakukan pada command line / shell pada sistem operasi linux/unix.

Untuk backup semua databases, menggunakan perintah:

Shell
1
# mysqldump -u root -p --all-databases > backup.sql

Untuk mengompres hasil backup-an kedalam file gz

Shell
1
# gzip backup.sql

Untuk merestore dari full backup

Shell
1
# mysql -u root -p < backup.sql

Jika melakukan restore pada full backup, maka perlu merestart service MySQLnya

Shell
1
# /etc/init.d/mysqld restart

atau

Restart Service MariaDB Server
